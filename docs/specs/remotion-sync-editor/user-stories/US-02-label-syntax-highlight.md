# US-02：編輯器中標籤語法突顯

**作為** 創作者
**我想要** 在編輯器中看到 `[labelName]` 以不同顏色或粗體顯示
**以便** 一眼辨識標籤位置，與內文區隔

## 輸入格式
- 使用者於編輯器輸入的文字
- 合法標籤語法（沿用 US-01 規則）

## 輸出格式
- 編輯器 UI：合法標籤以視覺樣式突顯（顏色或粗體，與內文明確區隔）
- 不影響文字輸入、游標移動、複製貼上行為

## 驗收條件
- [ ] 合法 `[labelName]` 在編輯器中以突顯樣式顯示
- [ ] 非法括號內容不被突顯
- [ ] 輸入、刪除、貼上時樣式即時更新
- [ ] 不破壞現有 TelePrompter 既有輸入體驗
- [ ] 視覺在淺色/深色主題下皆可辨識

## 依賴關係
- US-01（標籤語法解析器）

**優先級**：P1
**相關需求**：REQ-2、US-1
