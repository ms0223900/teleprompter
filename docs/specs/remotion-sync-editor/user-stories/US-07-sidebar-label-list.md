# US-07：側欄標籤數據清單

**作為** 創作者
**我想要** 在側欄看到每個標籤的預計秒數與總幀數
**以便** 判斷 Remotion 動畫要設計得複雜還是簡單

## 輸入格式
- `LabelTiming[]`（來自 US-06）

## 輸出格式
- 側欄 React 元件，表格或清單形式，顯示欄位：
  - `label` 名稱
  - `words`（字/詞數）
  - `seconds`（小數一位）
  - `frames`（整數）
  - `startFrame`（累積起點）

## 驗收條件
- [ ] 每個合法標籤顯示一列
- [ ] 順序與標籤在文稿中出現順序一致
- [ ] 秒數 > 警示閾值（例如 10 秒）時以視覺樣式提示
- [ ] 秒數 < 警示閾值（例如 1 秒）時以視覺樣式提示
- [ ] 空列表時顯示引導提示（例如「在文稿中插入 `[label]` 以開始」）
- [ ] RWD：在小螢幕下不破版

## 依賴關係
- US-06（即時重算 Hook）

**優先級**：P0
**相關需求**：REQ-6、US-2
