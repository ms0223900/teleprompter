# US-005：側欄「雙擊可編輯」說明文案

**作為** 影片編輯者  
**我想要** 在側欄一眼看到如何編輯標籤的提示  
**以便** 不必只靠 tooltip 才發現標籤名稱可編輯  

**優先級**：P1  
**相關功能**：Side Panel Label Management — 編輯 Label（US-001 鉛筆／雙擊互動）  

---

## 輸入格式

- 側欄已渲染標籤列表（至少一個合法標籤），或使用者開啟「?」說明／空列表使用指南

## 輸出格式

- 有標籤列表時：標題列下方顯示一行說明（繁中），明確提及「雙擊」，並附鉛筆圖示為替代操作
- 空列表 `UsageGuide` 與「?」`HelpPopover`：在適當段落補上相同行為說明，與列表區文案一致

## 驗收條件

- [x] 有列表時側欄常駐「雙擊可編輯」相關說明（非僅 `title`／`aria`）
- [x] 說明文字與實際互動一致（雙擊名稱、點擊鉛筆按鈕皆可進入編輯）
- [x] 不破壞現有 RWD（小螢幕側欄全寬時文字可換行、不溢出）
- [x] `UsageGuide` 與 `HelpPopover` 皆同步提及，方便空列表時預先理解

## 驗收說明

- 實作：`components/SyncSidebar.tsx`（列表上方提示列、`UsageGuide`、`HelpPopover`）

## 依賴關係

- **US-001**：鉛筆／雙擊編輯 UI 與行為
